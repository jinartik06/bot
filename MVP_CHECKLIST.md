# MVP Checklist

Сверка с исходным запросом.

## Сообщения

- Текст - есть.
- Голосовые - есть, через `faster_whisper`.
- Фото с подписью - есть.
- Фото без подписи - есть, создаётся карточка `Фото без подписи`.
- OCR текста на фото - есть через Tesseract, если он установлен.
- Альбом загруженных фото - есть.
- Удаление фото из альбома и локального хранилища - есть.
- Ссылки - есть, бот пытается подтянуть `<title>`.
- Пересланные сообщения - есть, обрабатываются как обычный текст/медиа от пользователя.

## AI-разбор

- Summary - есть.
- Разделение длинных мыслей на карточки - есть.
- Выделение задач - есть.
- Следующий шаг - есть.
- Автоопределение типа - есть.
- Автоопределение категории - есть.

## Экраны

- Первый экран - есть.
- `Как это работает` - есть.
- Главное меню - есть.
- Мысли - есть.
- Следующие шаги - есть.
- Поиск - есть.
- Архив - есть.
- Альбом - есть.
- Подробности карточки - есть.

## UX

- Ручная сортировка не нужна.
- Категории второстепенны и не вынесены в основное меню.
- Карточки короткие.
- На карточке только `Подробнее` и `Архивировать`.
- Длинные AI-анализы убраны из основной карточки.

## Проверка

```powershell
.\.venv\Scripts\python.exe -m unittest discover -s tests
.\.venv\Scripts\python.exe -m compileall -q src tests
```

Обе проверки проходят.

`startup_check` на текущем локальном `.venv` ожидаемо останавливается, потому что там Python 3.14. Для реального Whisper на Windows нужен Python 3.12 или Docker.
