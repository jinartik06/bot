# Гайд по боту

## Главная идея

Бот - это простое место для мыслей. Пользователь открывает Telegram, за несколько секунд отправляет текст, голосовое, фото, ссылку или пересланное сообщение и закрывает приложение. Сортировку и разбор делает AI.

## Первый экран

Текст `/start`:

> Привет.
>
> Это место для мыслей, задач и идей.
> Сюда можно быстро скинуть всё, что не хочется держать в голове.
>
> Что можно отправлять:
> • текст
> • голосовые
> • фото
> • ссылки
> • пересланные сообщения
>
> Что я сделаю:
> • разберу мысли
> • выделю задачи
> • сделаю короткое summary
> • предложу следующий шаг
>
> Можно прислать даже длинное хаотичное сообщение - AI сам разложит его на отдельные идеи.

Кнопки первого экрана:

- `➕ Добавить мысль`
- `💭 Мысли`
- `❓ Как это работает`

## Главное меню

- `💭 Мысли`
- `✍️ Продолжить мысль`
- `🔍 Поиск`
- `📎 Архив`
- `🖼 Альбом`
- `⚙ Настройки`

У администратора дополнительно видна кнопка `🛡 Админка`.

## Как бот обрабатывает сообщения

Текст:

- чистит шум;
- делает summary;
- выделяет задачи;
- предлагает следующий шаг;
- разбивает длинный текст на карточки.

Голос:

- скачивает голосовой файл;
- расшифровывает локально через `faster-whisper`;
- отправляет расшифровку в AI-разбор;
- создаёт одну или несколько карточек.

Фото:

- сохраняет файл в `data/photos`;
- использует подпись;
- отправляет изображение в vision-модель, если включён `PHOTO_VISION_ENABLED`;
- пытается распознать текст через Tesseract;
- добавляет в AI-разбор подпись, AI-описание фото и найденный OCR-текст;
- не придумывает визуальные детали, если vision/OCR недоступны;
- создаёт короткую карточку `Фото без подписи`, если подпись, AI-описание и OCR отсутствуют.

Продолжить мысль:

- показывает карточки с выделенными задачами и action items;
- у каждой карточки есть `Продолжить`, `Подробнее`, `Архивировать`;
- `Продолжить` позволяет дописать новый контекст к выбранной мысли и обновить её разбор.

Альбом:

- показывает все загруженные фото;
- отправляет фото с короткой подписью карточки;
- позволяет открыть подробности;
- позволяет удалить фото из альбома и локального хранилища, не удаляя саму карточку.

Ссылки:

- определяет URL;
- пытается подтянуть `<title>` страницы;
- отправляет URL и заголовок в AI-разбор.

## Формат карточки

Карточка короткая:

- тип записи;
- заголовок;
- summary;
- следующий шаг, если он есть.

Кнопки:

- `Подробнее`
- `Архивировать`

В подробностях показываются полный текст, summary, задачи, следующий шаг, категория, OCR-текст фото и теги.

## Настройки окружения

Голос:

```env
VOICE_TRANSCRIBER=faster_whisper
WHISPER_MODEL=small
WHISPER_DEVICE=cpu
WHISPER_COMPUTE_TYPE=int8
WHISPER_LANGUAGE=ru
```

Фото/OCR:

```env
PHOTO_STORAGE_DIR=data/photos
PHOTO_VISION_ENABLED=true
GROQ_VISION_MODEL=meta-llama/llama-4-scout-17b-16e-instruct
PHOTO_VISION_TIMEOUT_SECONDS=60
PHOTO_OCR_ENABLED=true
TESSERACT_BINARY=tesseract
PHOTO_OCR_LANGUAGE=rus+eng
PHOTO_OCR_TIMEOUT_SECONDS=30
```

На Windows для `faster-whisper` используйте Python 3.12. Python 3.14 не подходит для реальной загрузки native runtime.

## Проверка перед деплоем

```powershell
.\.venv\Scripts\python.exe -m unittest discover -s tests
.\.venv\Scripts\python.exe -m compileall -q src tests
.\.venv\Scripts\python.exe -m src.startup_check
```

Если `startup_check` пишет про Python 3.14 на Windows, код в порядке, но локальное окружение нужно пересоздать на Python 3.12 или запускать через Docker.
